The Brief

Problem → Solution

The Problem

Job and freelance applications are slow and guesswork-heavy. Most AI resume tools inflate: they invent metrics and write cover letters that collapse under a single follow-up question.

I wanted the opposite. A tool that gives a straight answer: am I a fit, where are the gaps, and how do I present my real experience better for this role, without overclaiming anything I would have to defend on a call.

The Solution

A full-stack AI app that analyzes a job description against a structured vault of real, documented experience and returns an honest fit score, a gap breakdown, and evidence-based resume suggestions.

The core constraint shaped everything: the tool cannot lie. Every suggestion must trace to something actually documented. That rule lives in the prompt layer, not just in good intentions.

"It highlights and reframes. It never fabricates."
System prompts require every claim to trace to documented experience, forbid inventing metrics, and distinguish proven LLM integration work from model building. Resume bullets and proposals are reframes of real experience, never inventions.

Capabilities

Features Built

🗂️
Resume Vault
Resume, work history, projects, education, and skills stored in Supabase as the single source of truth the AI reasons from.
🎯
Job Analyzer
Paste a job description; the full vault is assembled into a structured context block and analyzed against the role.
📊
Fit Report
Honest fit score, plain-spoken summary, must-have vs nice-to-have gaps, prioritized skills, and evidence-backed strengths.
✍️
Resume Optimizer
Bullet-level reframes plus surfacing of undersold experience already in the vault, each suggestion citing its evidence.
🕓
Analysis History
Every analysis saved and revisitable, to track applications and compare fit across roles over time.
💼
Freelance Mode
Evaluates Upwork gigs with a send-or-skip verdict framework, then drafts client proposals under strict voice and integrity rules.
🧩
Plain-English Capture
Brain-dump a side project in natural language; Claude structures it into clean fields for review before saving.
🔒
Secure Proxy
A Cloudflare Worker holds the API key as a secret, so it never ships in the public frontend bundle.
📦
Store Rich, Output Brief
The vault keeps full structured detail for accurate comparison; generated bullets apply smart brevity to what was actually produced.

Portfolio Value

Skills Demonstrated

Full-Stack Engineering
React Frontend to PostgreSQL Backend
Designed and built a complete full-stack application from zero — component architecture, a relational schema with row-level security, API integration, and a deployment pipeline. Every layer architected independently.
Prompt Engineering
Honest, Structured AI Output
System prompts constrained to fixed JSON shapes for reliable rendering, with integrity rules that forbid fabrication and enforce evidence-backed claims. The hard part of the build, and the part that makes it trustworthy.
Security Architecture
Cloudflare Worker API Proxy
The Anthropic API key is held as a Worker secret and never reaches the public frontend bundle. Row-level security enabled across all tables. A production-grade pattern for client-side AI products.
AI Product Design
Model Output to Usable Product
Turned raw model responses into a coherent product: assembled one consistent context block for every feature, parsed structured output defensively, and rendered fit reports, reframes, and proposals a user can act on.
